# Agent Prompt Patterns

Ready-to-use prompts for applying the negotiate skill in real time.

---

## Negotiation Preparation
```
Given this negotiation context: [describe situation, counterpart, objective]
Run the prepare-negotiation module. Output:
1. My best/acceptable/walkaway outcomes
2. Counterpart analysis (likely needs, fears, constraints)
3. Accusation audit (3-5 statements)
4. Opening statement
5. 5 calibrated questions
6. 2-3 black swan hypotheses
```

## Tactical Empathy Rewrite
```
Rewrite my reply using tactical empathy. Here is my draft response: [paste draft]
Context: [describe situation]

Rules:
- Open with a label or accusation audit, not a demand
- Replace demands with calibrated questions
- Use a no-oriented question instead of a pressuring yes-question
- Keep the same intent but change the emotional register
- Show me original vs. rewritten side-by-side
```

## Calibrated Question Generator
```
Given this negotiation context: [describe situation]
Generate 5 calibrated questions that:
1. Surface hidden constraints
2. Make the counterpart solve my problem with me
3. Test commitment without pressuring
4. Probe for black swans
5. Redirect from positional bargaining to interest-based discussion
For each question, explain what it's designed to surface.
```

## Accusation Audit Generator
```
I'm about to [describe your ask or proposal].
The counterpart is [describe who they are and what they care about].
Draft an accusation audit that preempts their likely objections.
Include 3-5 statements, ordered from most to least serious.
Show me how to deliver it (opening 2-3 sentences).
```

## Concession Detector
```
Here is my negotiation exchange so far: [paste conversation or key points]
Identify where I:
1. Conceded too early without getting something in return
2. Accepted their framing without questioning it
3. Missed an opportunity to mirror or label
4. Failed to test a soft "yes"
5. Could have used a calibrated question instead of a statement
For each finding, show what I should have done instead.
```

## Response to Difficult Message
```
I received this message in a negotiation: [paste message]
Context: [describe situation]
Help me respond using the negotiate skill:
1. What is this message really saying?
2. What emotions or pressures are behind it?
3. What should I mirror or label?
4. Draft a response using tactical empathy and calibrated questions
5. What should I NOT do?
```

## Black Swan Hunter
```
Here's my negotiation situation: [describe]
Something doesn't add up: [describe what's confusing]
Help me identify potential black swans:
1. What information would change everything if I knew it?
2. What behavior of theirs doesn't match their stated position?
3. What questions would surface these hidden factors?
4. Where should I look for this information beyond the negotiation table?
```

## Closing and Implementation Check
```
We've reached a tentative agreement: [describe what was agreed]
Help me:
1. Summarize the agreement using their language
2. Draft the Rule of Three confirmation sequence
3. Write a follow-up email that locks in commitments
4. Identify the 3 most likely places this agreement could fall apart
5. Suggest a check-in structure
```
